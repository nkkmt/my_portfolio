本のマッチングアプリ「Issatsu」

「選択肢が多すぎて、本を選べない」という課題を解決するためのマッチングアプリです。
楽天ブックスAPIを活用し、膨大な書籍データの中から直感的な操作で「次の一冊」に出会える体験を提供できるようにこだわりました。

制作期間

約2週間

使用技術

HTML / CSS / JavaScript

詳細説明

カードスワイプ機能

JavaScriptを用いて、Tinder風のスワイプUIを実装。
いいね／スキップを直感的に行えるよう、クリックだけではなく、フリック動作と滑らかなアニメーション、視覚的なフィードバックにより、楽しく本を選べるUXを追求しました。

マイリスト・スキップ履歴

スワイプ結果（いいね／スキップ）をLocalStorageに保存し、リロード後もデータが保持される仕組みを構築。
データはマイリストとスキップ履歴から確認できる機能を実装しました。

詳細な検索機能

「直感的な出会い」だけではなく、特定の条件で探したいニーズにも対応。
タイトル・著者名・ジャンルによる絞り込み機能を備え、検索結果から直接マイリストへ追加できる動線も確保しました。
